## get started
Easily install `@snappmarket/hooks` first, then use any hook you want in your components with fully
tree-shakable code.

```bash
yarn add @snappmarket/hooks
```

## usage
```javascript
import { useDebounce } from '@snappmarket/hooks';

const SampleComponent = props => {
  const [value, setValue] = useState('');
  const [debouncedValue] = useDebounce(value, 200);

  /**
   * Call api based on debounced value
   */
  useEffect(() => {
    // do something with debounce
  }, [debouncedValue]);
};
```
