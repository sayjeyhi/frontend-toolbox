## get started
Easily install `@snappmarket/hooks` first, then use any hook you want in your components with fully
tree-shakable code.

```bash
yarn add @snappmarket/hooks
```

## usage
```javascript
import { useElementScroll } from '@snappmarket/hooks';

const SampleComponent = props => {
  const elementRef = useRef();
  const handleScroll = () => console.log("load more");

  useElementScroll({
    reference: elementRef,
    callback: handleScroll,
    hasMore: true,
    threshold: 100
  });

  return (
    <div ref={elementRef}>
        // content goes here
    </div>
  );
};
```
