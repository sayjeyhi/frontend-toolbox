## get started
Easily install `@snappmarket/hooks` first, then use any hook you want in your components with fully
tree-shakable code.

```bash
yarn add @snappmarket/hooks
```

## usage
```javascript
import { useImageSize } from '@snappmarket/hooks';

const SampleComponent = props => {
  const [width, height] = useImageSize(url);

};
```

## useImageSize
A hook that gets image url and returns [width, height]

**Kind**: global function
**Returns**: array - width, height

| Param | Type | Description |
| --- | --- | --- |
| url | string | The image url |
