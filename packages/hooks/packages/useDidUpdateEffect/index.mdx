## get started
Easily install `@snappmarket/hooks` first, then use any hook you want in your components with fully
tree-shakable code.

```bash
yarn add @snappmarket/hooks
```

## usage
```javascript
import { useDidUpdateEffect } from '@snappmarket/hooks';

const SampleComponent = props => {
  /**
   * Do not run on render
   */
  useDidUpdateEffect(() => {
    // I will run only when my dependencies update
  }, [deps]);
};
```
