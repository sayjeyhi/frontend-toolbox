## get started

Easily install `@snappmarket/helpers` first, then use any helper you want in your components with fully
tree-shakable code.

```bash
npm i @snappmarket/helpers
```

## usage
```javascript
import { METHOD_NAME } from '@snappmarket/helpers';
```

## zeroPadding
adds starting zeros to numbers

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| number | number | number you want to add zero to its start |
| padding | number | count of zeros you want to add for a number to fix its length |
