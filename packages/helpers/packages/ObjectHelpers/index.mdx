## get started

Easily install `@snappmarket/helpers` first, then use any helper you want in your components with fully
tree-shakable code.

```bash
npm i @snappmarket/helpers
```

## usage
```javascript
import { METHOD_NAME } from '@snappmarket/helpers';
```

<a name="removeByKey"></a>

## removeByKey
returns an object without given key

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| haystack | object | object you want to remove key from |
| needle | string | key you want to remove from object |

<a name="flattenObject"></a>

## flattenObject
recursively flattens an object, properties keys would be combination of parent and sub-object keys

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| object | object | object you want to make it single dimensional |

<a name="safeObjectPropertyRead"></a>

## safeObjectPropertyRead
reads properties of an object safely and avoids to throw an error if property or object are undefined

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| object | object | object you want to get value from |
| key | string | key of the object you want to get it's value |
| defaultValue | any | default value of what you want if it was undefiend |

<a name="getNextProp"></a>

## getNextProp
returns the next property of given property in an object

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| object | object | object you want navigate through |
| key | string | name of the key you want to get next prop right after |

<a name="serializeObject"></a>

## serializeObject
serializes the properties of given object

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| object | object | object to be serialized by keys |

<a name="removeManyByKeys"></a>

## removeManyByKeys
returns an object without given keys

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| haystack | object | object you want to remove key from |
| needles | Array.&lt;string&gt; | keys you want to remove from object |

<a name="deepMerge"></a>

## deepMerge
merges two object in all levels

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| primaryObject | object | base object |
| secondaryObject | object | other object you want to merge with base object |
