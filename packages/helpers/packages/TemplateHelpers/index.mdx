## get started

Easily install `@snappmarket/helpers` first, then use any helper you want in your components with fully
tree-shakable code.

```bash
npm i @snappmarket/helpers
```

## usage
```javascript
import { METHOD_NAME } from '@snappmarket/helpers';
```

## makeShadow
returns a shadow suitable for css

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| hOffset | number \| string | horizontal offset of shadow, wil be multiplied in rem |
| vOffset | number \| string | vertical offset of shadow, wil be multiplied in rem |
| blur | number \| string | blur of the shadow, wil be multiplied in rem |
| spread | number \| string | spread of the shadow, wil be multiplied in rem |
| color | string | color of the shadow |
| inset | boolean | blur of the shadow, wil be multiplied in rem |
