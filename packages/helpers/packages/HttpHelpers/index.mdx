## get started

Easily install `@snappmarket/helpers` first, then use any helper you want in your components with fully
tree-shakable code.

```bash
npm i @snappmarket/helpers
```

## usage
```javascript
import { METHOD_NAME } from '@snappmarket/helpers';
```

## fetchWithTimeOut
fetches the request, if it takes to long, it will be timed out

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| url | string | the URL you want to fetch |
| options | object | the options such as get parameters, method etc... |
| timeout | number | timeout in ms to kill the request if it's taking too long |

<a name="universalCall"></a>

## universalCall
Call an api in universal way

**Kind**: global function

| Param | Type | Description |
| --- | --- | --- |
| url | string | the URL you want to fetch |
| method | string | method of the request e.g. POST |
| credentials | string | credentials to include the third party cookies or not e.g. same-origin |
| data | object | object of the data you want to send, not for get |
| headers | object | object of the request headers |
| params | object | URL parameters |
| jwtToken | string | the JWA token for end-points with authentication, Bearer prefix would be added automatically |
| rest | any | any additional options which is supported by the isomorphic fetch |
| timeout | number | timeout to kill the request if it's taking too long |
| allowedNoContent | boolean | flag to allow the request be handle even if there is no content |
