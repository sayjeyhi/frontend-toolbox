# Table
The table component accepts header, footer and data and renders them in given order, you can set the responsive classname, so the table will be responsive for medium viewports.

<br />

# Basic Usage

<Playground>
  <Table
    className="responsive"
    headers={['id', 'name', 'description', 'date']}
    data={[
      [1, 'foo', 'this is the description of the foo item', '2020-05-02'],
      [1, 'bar', 'this is the description of the bar item', '2020-05-03']
    ]}
    footers={['id', 'name', 'description', 'date']}
  />
</Playground>
